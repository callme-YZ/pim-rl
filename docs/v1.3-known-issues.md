## Issue 1: Resistive Term Numerical Instability 🔴 CRITICAL

**Status:** ⚠️ **Identified, Workaround Available**

### Symptom
Time evolution with resistivity (η > 1e-5) becomes unstable within a few steps:
- Energy explodes exponentially
- Max|ψ| grows from ~0.01 to >1e6 in <10 steps
- Occurs even with small time step (dt=1e-6)

### Root Cause
**Explicit treatment of resistive diffusion term** in symplectic integrator:
```python
psi_new = psi + 0.5 * dt * bracket - 0.5 * dt * eta * J
```

Resistive diffusion `-η·J` where `J ∝ ∇²ψ` is a **stiff term** requiring implicit time stepping for stability.

### Evidence
| Configuration | η | ν | dt | Result |
|---------------|---|---|----|--------|
| Pure Hamiltonian | 0 | 0 | 1e-6 | ✅ Stable (E/E0 = 1.000) |
| Only viscosity | 0 | 1e-4 | 1e-5 | ✅ Stable (E/E0 = 1.000) |
| Only resistivity | 1e-4 | 0 | 1e-5 | ❌ Unstable (step 1) |
| Both (weak) | 1e-6 | 1e-6 | 1e-5 | ⚠️ Stable but drifts (E/E0 = 1.44 @ 10 steps) |

**Interpretation:**
- Viscosity (`-ν·∇²ω`) is stable even explicit (acts on ω, not ψ directly)
- Resistivity (`-η·∇²ψ`) is unstable explicit (stiff diffusion on evolved variable)

### Workaround (Immediate)
**Use η = 0 or η < 1e-6:**
```python
solver = HamiltonianMHD(grid, dt=1e-5, eta=0.0, nu=1e-4)
# or
solver = HamiltonianMHD(grid, dt=1e-5, eta=1e-7, nu=1e-4)
```

**Limitations:**
- Cannot simulate realistic resistive MHD (typical η ~ 1e-4 - 1e-6)
- Energy drift increases with weak resistivity

### Solution (Future Work)
**Option A:** Implicit-explicit (IMEX) time stepping
- Treat resistive term implicitly
- Keep Hamiltonian terms explicit
- Requires solving linear system each step

**Option B:** Operator splitting
- Hamiltonian evolution (symplectic)
- Resistive diffusion (implicit or Crank-Nicolson)
- Alternating steps

**Option C:** Use built-in scipy integrators
- `scipy.integrate.solve_ivp` with stiff solver (BDF, Radau)
- Lose symplectic structure but gain stability

**Recommended:** Option A (IMEX) — preserves Hamiltonian structure while handling stiffness.

### Implementation Plan
1. Add implicit Poisson solve for resistive update:
   ```python
   # Instead of: psi_new = psi - dt * eta * J(psi)
   # Solve: psi_new - dt * eta * J(psi_new) = psi
   # → (I + dt * eta * Lap) psi_new = psi
   ```
2. Use same GMRES framework as Poisson solver
3. Test CFL condition: `dt < dr² / (2η)`

**Effort estimate:** 4-6 hours (solver modification + testing)

## Issue 2: Axis Singularity ✅ FIXED

**Status:** ✅ **Resolved** (Commit: 2708f3f)

### Problem
Division by zero at r=0 in current density:
```python
Delta_star_psi = d2psi_dr2 + d2psi_dtheta2 / r²  # r=0 → NaN
```

### Solution
Applied L'Hôpital's rule + axisymmetry:
```python
if r == 0:
    Delta_star_psi[0, :] = d2psi_dr2[0, :]  # d²ψ/dθ² = 0 at axis
else:
    Delta_star_psi[r, :] = d2psi_dr2[r, :] + d2psi_dtheta2[r, :] / r²
```

**Validation:** No NaN, J_phi matches analytical solution (error < 0.2%)
